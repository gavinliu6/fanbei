## 项目简介

作为一名简中程序员，译者在平时工作学习中遇到优质的英文资料时，习惯把它们翻译为中文以加强学习效果。为了集中管理这些翻译内容，译者便创建了该项目，起名为「翻贝」。项目主站使用 [VitePress](https://vitepress.dev/zh/) 搭建，部署在 [Netlify](netlify.com) 上。
